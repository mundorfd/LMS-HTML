Color-Coded Content Boxes

Multi-colored bordered boxes for organizing different types of content

Preview:

Primary Content

  • First important point
  • Second key detail
  • Third essential item

Secondary Content

  • Supporting information
  • Additional details
  • Related concepts

Ingredients:

  • border-radius: 8px
  • padding: 1.5rem
  • background: white
  • border: 2px solid [color-variable]
  • heading color: matches border color
  • heading size: 14pt, bold
  • content size: 12pt, line-height 1.6

HTML/CSS Code:

<div style="background: white; border: 2px solid #d73f09; padding: 1.5rem; border-radius: 8px;">
    <h3 style="color: #d73f09; font-size: 14pt; margin: 0 0 1rem 0; font-weight: bold;">Your Heading</h3>
    <div style="font-size: 12pt; line-height: 1.6;">
        <ul>
            <li>First point</li>
            <li>Second point</li>
            <li>Third point</li>
        </ul>
    </div>
</div>

Suggested Uses:

  • Course learning objectives and outcomes
  • Module or lesson content sections
  • Key concept explanations
  • Assignment requirements and rubrics
  • Resource and tool listings
Color Palette: #d73f09 (primary red), #4A773C (green), #00859B (teal), #FFB500 (yellow), #006A8E (blue)
