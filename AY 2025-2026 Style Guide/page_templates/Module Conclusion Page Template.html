Content Writing Guidelines:

  • Key Concept Questions: Write questions that test understanding, not just recall
  • Question Complexity: Progress from basic concepts to application and analysis
  • Learning Outcome Alignment: Each question should map to specific module learning outcomes
  • Self-Assessment Focus: Questions should help students identify knowledge gaps
  • Forward Connection: Mention how current concepts connect to upcoming material
  • Support Resources: Always direct students to help resources and discussion forums

Suggested Uses:

  • End-of-module summary and review pages
  • Self-assessment checkpoints before major assignments
  • Pre-exam review and preparation pages
  • Module closure with forward-looking connections
  • Learning outcome verification and student reflection
  • Standardized module navigation and completion confirmation
