Placeholder Guide:

  • [COURSE-HEADER-IMAGE-URL]: Replace with course-specific header image

  • [LEARNING OUTCOME 1-4]: Specific, measurable learning objectives
  • [COURSE LEARNING OUTCOME ALIGNMENT]: How module outcomes connect to course outcomes
  • [DECORATIVE-ICON-URL]: Small decorative image for learning outcomes box
  • [TASK-LIST-IMAGE-URL]: Relevant image for task list section
  • [TASK LIST IMAGE ALT TEXT]: Descriptive alt text for task image
  • [IMAGE CAPTION TEXT]: Caption describing the image content
  • [Module #: Learning Materials]: Replace with actual module number and link text
  • [LEARNING-MATERIALS-URL]: Canvas URL to learning materials page
  • [DISCUSSION-URL]: Canvas URL to discussion forum
  • [LAB-URL]: Canvas URL to lab instructions
  • [QUIZ-URL]: Canvas URL to quiz or assignment
  • [HOMEWORK-URL]: Canvas URL to homework assignment
  • [MUDDIEST-POINT-URL]: Canvas URL to muddiest point discussion
  • [MODULES-URL]: Canvas URL back to modules page
  • [RETURN-TO-MODULES-IMAGE-URL]: Navigation button image URL

Suggested Uses:
