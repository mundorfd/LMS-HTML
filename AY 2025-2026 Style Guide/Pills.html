Ingredients:

  • class: "pill" (Canvas built-in class)
  • semantic structure: Standard <ul> and <li> elements
  • responsive: Horizontal on desktop, vertical list on mobile
  • font-size: 18pt (customizable with span styling)
  • link styling: Optional - can mix links and plain text
  • color enhancement: Add OSU brand colors to links

HTML/CSS Code:

<ul class="pill">
    <li><span style="font-size: 18pt;"><a href="your-link" style="color: #d73f09; text-decoration: none;">Linked Item</a></span></li>
    <li><span style="font-size: 18pt;">Plain Text Item</span></li>
    <li><span style="font-size: 18pt;">Another Item</span></li>
    <li><span style="font-size: 18pt;">Final Item</span></li>
</ul>

Suggested Uses:

  • Course module and week navigation
  • Chapter or unit selection
  • Assignment category organization
  • Course section quick links
  • Topic overview navigation
Canvas Class: Uses Canvas built-in "pill" class for consistent styling across the platform

Size Guidelines:

  • 14pt: Content tags, categories, small groupings
  • 12pt: Minimal use - very fine categorization only

Color Usage Guide:

  • Beaver Orange (#d73f09): Primary links, main actions, current/active items
  • Pine Stand Green (#4A773C): Secondary links, resources, examples
  • Star Canvas Blue (#003b5c): Information categories, labels, metadata
  • Stratosphere Blue (#006A8E): Process steps, workflows
  • Gray (#666): Inactive or future items

Code Templates:

<!-- Colored Links -->
<li><span style="font-size: 16pt;"><a href="your-link" style="color: #d73f09; text-decoration: none;">Primary Link</a></span></li>
<li><span style="font-size: 16pt;"><a href="your-link" style="color: #4A773C; text-decoration: none;">Secondary Link</a></span></li>

<!-- Colored Text Labels -->
<li><span style="font-size: 14pt; color: #003b5c; font-weight: bold;">Category Label</span></li>

<!-- Inactive/Future Items -->
<li><span style="font-size: 16pt; color: #666;">Coming Soon</span></li>

Suggested Uses:

  • Color-coded course navigation by content type
  • Progress indication (active vs. inactive items)
  • Action vs. information distinction
  • Content categorization and tagging
  • Visual hierarchy in complex navigation
Accessibility Note: Color coding is supplemented with clear text labels and link styling to ensure accessibility compliance

Application Guidelines:

  • Course Navigation: Use larger fonts (16-18pt) with primary colors for active sections
  • Content Categories: Medium fonts (14pt) with different colors for each category type
  • Resource Links: Consistent secondary color (Green) for all external/support links
  • Metadata Tags: Smaller fonts (12pt) with muted colors for learning objectives, difficulty levels
  • Progress Indication: Use color intensity to show completed vs. upcoming content

Best Practices:

  • Limit Items: Keep 3-7 items per pill list for optimal usability
  • Consistent Spacing: Use uniform font sizes within each navigation level
  • Clear Labeling: Make link text descriptive and action-oriented
  • Mobile Testing: Remember pills become vertical lists on mobile devices
  • Color Meaning: Establish and maintain consistent color meanings across the course
Usage Tip: Pills work best for horizontal navigation and tagging. For complex multi-level navigation, combine with other components like your existing card layouts or definition lists
