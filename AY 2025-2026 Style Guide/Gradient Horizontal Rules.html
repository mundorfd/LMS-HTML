Beaver Orange Gradient Rule

Primary brand color section divider with fade effect

Preview:

This is some sample content above the horizontal rule.

This is some sample content below the horizontal rule, showing how it elegantly separates sections.

Ingredients:

  • border: none (removes default styling)
  • height: 1px (thin, elegant line)
  • background: linear-gradient(to right, transparent, #d73f09, transparent)
  • margin: 40px 0 (generous spacing above and below)
  • color: #d73f09 (Beaver Orange)
  • effect: fades from transparent to color and back

HTML/CSS Code:

<hr style="border: none; height: 1px; background: linear-gradient(to right, transparent, #d73f09, transparent); margin: 40px 0;">

Suggested Uses:

  • Major section breaks in long content
  • Separating different topics or modules
  • Creating visual breaks in assignment instructions
  • Dividing course introduction from main content
  • Separating theoretical content from practical applications
