Sectioned Content Boxes

Structured content containers with colored headers and bordered sections

Preview:

Section Title

This is the main content area of the sectioned box. It provides a clear visual hierarchy with the colored header and bordered content area.

  • List items work well in this format
  • Multiple paragraphs are supported
  • Links and other content elements integrate seamlessly

Alternative Color

The same structure works with different OSU brand colors for variety and visual organization.

Ingredients:

  • Header background: #d73f09 (Beaver Orange) or #008096 (Teal)
  • Header text: White (#ffffff)
  • Border: 3px solid matching header color
  • Content background: White
  • Text color: #423e3c (Canvas regular text)
  • Font: Arial, sans-serif (content), Georgia, serif (headers)
  • Border-radius: 6px on header, matching bottom corners

HTML/CSS Code:

<div style="margin-bottom: 40px; font-family: Arial, sans-serif; line-height: 1.6; color: #423e3c;">
    <h2 style="color: #ffffff; background-color: #d73f09; font-size: 14pt; margin: 0; padding: 12px 20px; border-radius: 6px 6px 0 0; font-family: Georgia, serif;">
        <span style="font-size: 20pt;">Your Section Title</span>
    </h2>
    <div style="padding: 25px; border: 3px solid #D73F09; border-top: none; border-radius: 0 0 6px 6px; margin-top: 0; background: white;">
        <p>Your content goes here. This can include paragraphs, lists, links, and other elements.</p>
        <ul>
            <li>List item 1</li>
            <li>List item 2</li>
        </ul>
    </div>
</div>

Color Variations:

  • Beaver Orange: #d73f09 (primary OSU color)
  • Teal: #008096 (secondary accent color)
  • Pine Stand Green: #4A773C (for nature/sustainability content)

Suggested Uses:

  • Module learning outcomes and objectives
  • Assignment instructions and requirements
  • Course policies and procedures
  • Task checklists and to-do items
  • Submission guidelines and rubrics
  • Resource collections and reading lists

Accessibility Features:

  • High contrast between header text and background (AA 2.1 compliant)
  • Semantic heading structure maintained
  • Clear visual hierarchy with color and typography
  • Canvas dark mode compatible with explicit background colors
Canvas Compatibility: Works in Canvas Rich Content Editor with inline CSS
Dark Mode: Compatible with explicit white backgrounds
