Ingredients:

  • circle size: 80px Ã— 80px
  • border-radius: 50% (perfect circle)
  • background colors: OSU brand colors
  • text color: white
  • font-size: 24px for numbers, 14pt for labels
  • font-weight: bold
  • margin: 0 auto 1.5rem auto (centers and spaces)
  • display: flex with center alignment
  • grid layout: Canvas Flexbox Grid system
  • responsive classes: col-xs-12 col-md-4 (3 columns on medium+ screens)
  • content-box: Canvas wrapper for proper spacing
  • grid-row: Canvas row container

Suggested Uses:

  • Step-by-step process instructions
  • Numbered learning objectives
  • Sequential module or lesson navigation
  • Multi-part assignment components
  • Categorized content with visual hierarchy
